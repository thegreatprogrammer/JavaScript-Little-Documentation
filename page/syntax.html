Primeros pasos en JavaScript

Los programas JavaScript pueden ser insertados en un documento HTML con ayuda de la etiqueta <script>

La manera más recomendada de trabajar con JavaScript es alojando el código en un archivo externo, lo cual se logra colocando la siguiente línea de código:
<script>

JavaScript es un lenguaje interpretado, no compilado. Significa que sólo se necesita el bloc de notas para programar un archivo .js, y vincularlo a una página para ver los resultados en un navegador.

Aunque no es imprescindible, es una buena práctica que los enunciados terminen en un punto y coma (;). Si éste no existe, el intérprete de JavaScript considerará un salto de línea como el final de un enunciado

La sintaxis de los comentarios depende del número de líneas:
//...esto es un comentario de una línea

/*..esto es un comentario multilínea*/


Los bloques de código son un conjunto de enunciados que deben interpretarse como una secuencia completa —como por ejemplo una función—. Se indican con llaves ({}).

Variables

Originalmente, en Javascript solo existía una forma de declarar variables. Se hacía con la palabra clave "var" seguida del nombre de variable a declarar (o variables separadas por comas).
var variable;
Esta variable podía ser de ámbito global o local. Su ámbito depende del lugar donde se declara pero las posiblidades eran bien limitadas. Para hacer una variable global la teníamos que declarar simplemente fuera del cuerpo de una función. En cambio, las variables locales eran declaradas dentro de una función y solo eran accesibles dentro del código de esa función.

Para solucionar todo esto en Javascript ES6 se ha implementado alguna forma nueva de declarar variables que ahora será la más indicada en la mayoría de los casos. Se trata de la construcción "let".

La clave de let es que restringe su existencia al ámbito donde ha sido declarada (cualquier ámbito expresado con unas llaves).
let newVar = 150;


Otra nueva forma de crear variables es "const". Ésto nos crea una variable donde su valor es inalterable a lo largo del código.

Por convección se acostumbra a definir en mayúscula los nombres de las constantes, para una mejor identificación
const NEW = 10;

Tipos de datos

Javascript es un lenguaje con tipado débil dinámico, es decir, las variables no tienen un tipo específico y es posible comparar datos de diferentes tipos produciéndose la conversión implícita de los valores al realizar la comparación.

Es posible utilizar typeof, una instrucción pensada de forma específica para conocer el tipo del valor de una variable. Funciona perfectamente con los tipos primitivos y con las funciones.

console.assert(typeof true === 'boolean');

Tipos primitivos


Javascript tiene seis tipos primitivos: Sin definir (undefined), Nulo (null), Lógicos (boolean), Numérico (number), Cadena (string), Símbolo (symbol). Todos los demás tipos son objetos (Object): Array, Date, Promise, etc.

Operadores

Los operadores son símbolos con los que, como su nombre indica, se pueden realizar operaciones con variables, valores constantes o variables y valores constantes. Hay muchos tipos de operadores, pero para esta introducción voy a centrarme en los más comunes:

Los operadores son símbolos con los que, como su nombre indica, se pueden realizar operaciones con variables, valores constantes o variables y valores constantes. Hay muchos tipos de operadores, pero para esta introducción voy a centrarme en los más comunes:
  • Operadores aritméticos
  • Operadores comparativos
  • Operadores lógicos

Aritméticos


Los operadores aritméticos son: adición, sustracción, multiplicación, división, módulo.

Comparativos


Como su nombre indica, estos operadores se emplean para realizar comparaciones entre los valores de las variables. Destacan los símbolos > (mayor que), < (menor que), == (igual a), != (distinto de), >= (mayor o igual a), <= (menor o igual a), === (idéntico a).

Lógicos


Los operadores lógicos amplian las posibilidades de establecer condiciones. Se presenta el operador de negación (!), operador "y" (&&), operador "o" (||).
