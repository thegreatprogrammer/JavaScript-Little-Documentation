Introducción

Una función no es más que un bloque de enunciados que componen un comportamiento que puede ser invocado las veces que sea necesario. Una función de JavaScript presenta el siguiente aspecto: 
function nombre_de_la_función(){
	…enunciados a ejecutar…
}
Para ejecutar la función posteriormente no hay más que invocar su nombre en cualquier momento y desde cualquier parte de un código, con una excepción: la función debe haber sido definida anteriormente. Así, una función llamada "obtenerNombre" se invocaría en cualquier parte del programa de la siguiente forma: 
obtenerNombre(); //Invocando función

Funciones de expresión

Una expresión con una función es: 
let sum = function(a,b) {
	return a + b;
};
Estas funciones se caracterizan por ser declaradas dentro de una variable; la principal diferencia entre este tipo de función y las funciones generales (declarativas) radica en que puede ser utilizada sí y solo sí ha sido declarada; lo cual no ocurre con las funciones declarativas, que aún antes de ser declaradas pueden ser invocadas sin inconveniente.

Funciones flecha

Las "arrow functions" de ES6 son una nueva manera de expresar las funciones de siempre, de un modo resumido y con algunas características nuevas. Su sintaxis es simple: se sustituye la palabra clave function por el símbolo de la flecha, como se puede ver en el siguiente código: 
let mifuncion = () => {
  //código de la función
}
